addPackageToQueue( package, queue )

Add a given package to the given queue. If package already in the queue then it will not be added twice and return a positive answer.

PARAM String - the package to be added to the queue. PARAM String - the queue to add the package to.

RETURN boolean - true if package added/exists to/in install queue, otherwise false.
